# Semantic Textual Similarity (STS)
## 과제 목표
- 두 개의 한국어 문장을 입력받아 두 문장의 의미적 유사도 출력
- regression task ( 0 ≤ target ≤ 5)
- FAST API 웹 프레임워크를 사용하여 두 문장의 유사도 점수 출력하는 Server side Code
